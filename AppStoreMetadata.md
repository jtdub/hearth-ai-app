# App Store Metadata — Hearth AI

## App Name
Hearth AI

## Subtitle
Private AI Chat, On-Device

## Description
Hearth AI brings powerful language models directly to your iPhone. Every conversation is completely private — all AI processing happens on your device with no internet required.

**Truly Private**
No data collection. No cloud servers. No tracking. Your conversations stay on your device and are protected by iOS encryption.

**Powerful Models**
Browse and download open-source AI models from Hugging Face Hub. Choose from curated recommendations or search thousands of available models.

**Works Offline**
Once a model is downloaded, you can chat anywhere — no Wi-Fi or cellular connection needed.

**Features**
- On-device AI inference powered by llama.cpp
- Browse and download GGUF models from Hugging Face Hub
- Conversation history with per-chat settings
- System prompt customization
- Adjustable temperature and sampling parameters
- Background downloads with pause/resume
- Automatic device memory gating
- Thermal monitoring to protect your device
- Dark mode support

## Keywords
AI, chat, private, offline, local, LLM, GPT, assistant, on-device, language model

## Category

## App Review Notes
Hearth AI is a local AI chat application. Model files are neural network weight data (GGUF format) downloaded from Hugging Face Hub. These are not executable code. All inference runs on-device using the open-source llama.cpp library. No user data leaves the device.

The app downloads model files ranging from ~300MB to ~4GB. These files are publicly available machine learning model weights. No authentication or account is required.

## Privacy Nutrition Label
Data Not Collected — Hearth AI does not collect any data from users.

## Export Compliance
The app uses HTTPS for downloading model files from Hugging Face Hub. This qualifies for the encryption export regulations exemption (ERN) as it uses only standard OS-provided encryption (URLSession/TLS). No custom encryption is implemented.

Self-classification: Exempt (uses only standard OS encryption for HTTPS)
